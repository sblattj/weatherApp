# Weather

## Set up Rails

## Clone Repo

```
git clone https://github.com/sblattj/weatherApp.git
```

## CD into weatherApp and starts Rails server

```
cd weatherApp
rails s
```

## Go to localhost:3000 on your machine
```
Browse Chrome -> http://localhost:3000
```
